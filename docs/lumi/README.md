# Support Site

Static support and privacy pages for App Store Connect / GitHub Pages.

## Files
- `index.html` - support page
- `privacy.html` - privacy policy page

## Suggested GitHub Pages Setup
1. Push this repo to GitHub.
2. Open repository settings on GitHub.
3. Go to `Pages`.
4. Set:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main`
   - `Folder`: `/docs`
5. After GitHub Pages is live, your URLs will look like:
   - `https://odukle.github.io/CineVerse/support/`
   - `https://odukle.github.io/CineVerse/support/privacy.html`

If you later move this to a dedicated site repo or custom domain, update the App Store URLs there.
